# SteganographierGUI
将文件隐写进MP4文件中

v1.0.1 修复了无法隐写ZIP格式文件的bug
v1.0.2 新增隐写为mkv文件的逻辑，引入了一下三个工具用于处理mkv文件
```
.\tools\mkvextract.exe
.\tools\mkvinfo.exe
.\tools\mkvmerge.exe
```
